⚡ Sparx Solver v3
Sparx Solver v3 is a powerful offline maths solver with an optional AI Mode for advanced explanations.
It runs fully in your browser, works offline, and supports everything from basic arithmetic to advanced algebra, matrices, and calculus.

🚀 Features
✔ Offline Maths Engine (No API Key Needed)
Powered by Math.js, the solver can handle:

Arithmetic

Algebra

Linear & quadratic equations

Expression simplification

Fractions

Trigonometry

Vectors

Matrices

Complex numbers

Derivatives

Integrals

Equation solving

All offline and instant.

🤖 Optional AI Mode (User‑Provided API Key)
AI Mode activates only when the user clicks AI Mode and enters their own API key.

Supports:

Step‑by‑step explanations

Word problems

Complex reasoning

Advanced algebra and calculus

AI Mode is optional — the app works without it.

🌓 Dark / Light Mode
Toggle between dark and light themes instantly.

📸 Screenshot Upload
Upload an image of a maths question.
The image is displayed so you can manually type the question.

🎯 Answer‑Only Mode
When enabled, only the final answer is shown.
Steps/explanations are hidden.

📦 How to Use
Open sparx-solver.html in any browser.

Type a maths question (e.g., 2x + 3 = 11, integrate(x^2, x)).

Click Solve for offline solving.

Click AI Mode for AI explanations (requires API key).

Toggle dark/light mode as needed.

Enable Answer‑Only Mode for quick checking.

Upload a screenshot if needed.

🔐 AI API Key (Optional)
AI Mode requires your own API key (OpenAI, Groq, DeepSeek, Gemini, etc.).

Your key:

Is stored only in your browser memory

Is never uploaded

Is never saved to a server

If you don’t want AI Mode, ignore this feature.
